# MCQ Quiz Analysis Project

This project analyzes MCQ quiz responses using Python and Streamlit.

## 📊 Features
- Department-wise performance analysis
- College-wise comparison
- Data visualization using charts
- Interactive dashboard using Streamlit

## 📁 Project Files
- app.py → Streamlit app
- analysis.py → Data processing
- responses.csv → Student responses
- answers.csv → Correct answers
- quiz_analysis.xlsx → Output report

## ▶️ How to Run
1. Install libraries:
   pip install streamlit pandas matplotlib openpyxl

2. Run the app:
   streamlit run app.py

## 💻 Technologies Used
- Python
- Pandas
- Matplotlib
- Streamlit
